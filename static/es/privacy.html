Política de Privacidad

Última actualización: abril 2026

Introducción

RealText ("nosotros") se compromete a proteger tu privacidad. Esta política explica cómo manejamos la información cuando utilizas nuestro sitio web y servicio de análisis de textos IA en realtext.org.

Datos del análisis textual

No almacenamos tu texto. Cuando envías un texto para análisis, se procesa en la memoria del servidor y se descarta inmediatamente al completar el análisis. Nunca guardamos, registramos ni compartimos el contenido enviado.

Datos que recopilamos

Recopilamos los datos mínimos necesarios para operar el servicio:

  • Contadores anónimos de uso: Monitorizamos el número de análisis por dirección IP al día para aplicar el límite diario gratuito. No se almacena contenido textual.
  • Registros del servidor: Los registros estándar del servidor web pueden contener direcciones IP, tipo de navegador y marcas de tiempo. Se utilizan exclusivamente para seguridad y depuración.

Lo que NO recopilamos

  • No requerimos creación de cuenta ni registro
  • No recopilamos correos electrónicos, nombres ni identificadores personales
  • No usamos cookies de seguimiento o publicidad
  • No compartimos datos con terceros con fines de marketing
  • No almacenamos ni registramos el texto enviado para análisis

Servicios de terceros

Nuestro análisis textual puede utilizar APIs de IA de terceros (como OpenAI o Anthropic) para generar sugerencias de humanización. En tal caso, tu texto se envía a estos servicios solo para procesamiento. Estos servicios tienen sus propias políticas de privacidad.

Seguridad de datos

Utilizamos medidas de seguridad estándar de la industria para proteger nuestra infraestructura, incluyendo conexiones cifradas (HTTPS), servidores con cortafuegos y retención mínima de datos.

Tus derechos

Dado que no recopilamos datos personales ni almacenamos contenido de usuarios, los datos a acceder, modificar o eliminar son mínimos. Para preguntas sobre tus datos, contáctanos en la dirección indicada a continuación.

Cambios en esta política

Podemos actualizar esta política de privacidad periódicamente. Los cambios se publicarán en esta página con una fecha de "Última actualización" actualizada.

Contacto

Para preguntas sobre privacidad, contáctanos en: privacy@realtext.org
